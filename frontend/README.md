# HAK Burgers - POS System

Welcome to the HAK Burgers POS System! This project is a web application designed to modernize and streamline the operations of the "HAK Burgers" shop.

## Project Overview

The application includes features for:
- Stock Management: Add, update, and delete food items. Manage categories and expiration dates.
- Order Management: Handle orders with cart functionality, price calculations, discounts, and PDF receipt generation.
- Customer Management: Manage customer details and track orders.
- Report Generation: Generate monthly and annual sales reports, view top customers, and track food item counts.

## Technologies Used

### Frontend

- HTML: Structure of the web pages
- CSS: Styling and layout
- JavaScript: Interactivity and dynamic content

### Backend

- NodeJS: Used as the backend runtime to handle APIs, business logic, and connect to the MongoDB database.

### Database

- MongoDB

## Setup Instructions

1. Configure `backend/.env` (you can copy `backend/.env.example`).
2. Start backend with `npm start` inside the `backend` directory.
